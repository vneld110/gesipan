Chapter 03. 어트리뷰트 다루기

  • 엘리먼트의어트리뷰트값을가져오고, 설정하기- attr()
  • 엘리먼트집합들을한번에처리하기-attr( { attributeName : value, attributeName : value, .... } )
  • 엘리먼트의어트리뷰트값을콜백함수에서얻어와서설정하기-attr(attributeName, function(index))
  • 엘리먼트의어트리뷰트값제거하기-removeAttr(attributeName)
  • 엘리먼트의클래스명으로스타일입히기-addClass( function(index, className) )
  • 엘리먼트로부터클래스제거하기-removeClass( className )
  • 엘리먼트에해당클래스가있으면제거하고, 없으면추가하기-toggleClass(className)
  • 엘리먼트에클래스존재여부확인하기-hasClass(className)
  • 엘리먼트의태그내부의HTML 콘텐츠얻어오기-html()
  • 엘리먼트의태그내부의HTML 콘텐츠변경하기-html(contents)
  • 폼엘리먼트에서value 어트리뷰트값가져오고, 설정하기-val()
