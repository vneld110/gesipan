Chapter 04. DOM 탐색하기

  • DOM 탐색 메소드에 대해서
  • 탐색한 엘리먼트들에 기준이 되었던 엘리먼트들 추가하기-add()
  • 셀렉트된 엘리먼트들 중에서 조건에 맞는 엘리먼트 얻기-filter()
  • 셀렉트된 엘리먼트들 중 표현식을 만족하지 않는 엘리먼트 얻기-not()
  • 셀렉트된 엘리먼트들 중 원하는 범위의 엘리먼트만 반환하기-slice()
  • 해당 엘리먼트의 자식 엘리먼트들 찾기-children()
  • 해당 엘리먼트 내부 탐색 및 내용 변경하기-contents()
  • 해당 엘리먼트 뒤에 위치하는 형제 엘리먼트 찾기-next()
  • 해당 엘리먼트 뒤에 있는 모든 엘리먼트 찾기-nextAll()
  • 해당 엘리먼트 앞에 있는 형제 엘리먼트 찾기 -prev()
  • 해당 엘리먼트 앞에 있는 모든 형제 엘리먼트 찾기-prevAll()
  • 해당 엘리먼트와 같은 노드 위치의 모든 엘리먼트 찾기-siblings()
  • 해당 엘리먼트의 모든 부모 엘리먼트 찾기-parents()
  • 해당 엘리먼트의 가장 인접한 부모 엘리먼트 찾기-parent()
  • 셀렉트된 엘리먼트들 중에서 자손 엘리먼트 찾기-find()
  • 셀렉트된 엘리먼트들의 내용을 변경하기-map()
  • 탐색한 엘리먼트들에 기준이 되었던 엘리먼트들 추가하기-andSelf()
  • 메소드 체이닝(chaning) 중에 최초 엘리먼트로 돌아가기-end()
